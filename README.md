## A Machine Learning Regression Model to Predict the Luminescent Quenching Concentration of Inorganic Phosphors
This repository hosts a catalog containing compositional and structural properties of Tb³⁺ host matrices, constructed using a database of chemical elements.

The catalog is well suited for machine learning applications, particularly regression problems in materials science.
